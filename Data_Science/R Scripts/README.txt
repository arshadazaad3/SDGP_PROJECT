The functionality of the R Script (StructuringData.R) is to fetch data from the BBC data folder where the data are
stored as a .txt files. There are 6 folders in the BBC data set (business, entertainment, politics, sport, tech).
And in each folder there are many text files. those text files has a name (Name of the article), a content (Content of
the article).
So the functionality of this R script is to is to create a dataset with the following information:
   - Name of the article
   - Content of the article
   - Category of the article
